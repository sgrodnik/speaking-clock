# Speaking Clock

An ascetic, minimalist web-based timer that periodically announces the current time using the browser's built-in Text-to-Speech (SpeechSynthesis) engine.

[Demo App](https://sgrodnik.github.io/speaking-clock/)

## Features

- **Total Minimalism:** Pure HTML/JS with almost no CSS and zero external dependencies.
- **Multilingual Support:** Announces time in Russian (RU), English (EN), and Spanish (ES) with proper language-specific phrasing.
- **Precision Syncing:** Option to sync announcements exactly on the minute mark (`00` seconds) instead of a simple interval delay.
- **Countdown Timer:** High-precision visual countdown displaying the time remaining until the next voice announcement.
- **Privacy & Offline First:** Runs entirely inside the browser without sending your data to external servers.
